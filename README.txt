Cryptocurrency Price Prediction Using Deep Learning
This is just the model of deep learning. There is no UI for this model.

1. Open 'Cryptocurrency_Price_Prediction.ipynb' in Google Colab.
2. Upload dataset 'Binance_BTCUSDT_d(12_2017_12_2022).csv' before run the code.
3. After the code successfully runned, a new file/dataset will be created, 'predicted output.csv'
4. The last 30 days from the new file/dataset are the predicted bitcoin close price.
5. Open 'cpp visualization.pbix' to see the project dashboard.
